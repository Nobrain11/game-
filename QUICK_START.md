# 🎮 ASTRALIS RPG - Quick Start (5 Minutes)

## Step 1: Get Bot Token (2 min)

1. Open Telegram
2. Search: `@BotFather`
3. Send: `/newbot`
4. Follow prompts
5. Copy your **BOT_TOKEN**

## Step 2: Configure (1 min)

Edit `.env`:
```
BOT_TOKEN=paste_your_token_here
```

## Step 3: Install & Run (2 min)

```bash
npm install
npm start
```

You should see:
```
✅ Bot is running!
```

## Step 4: Test in Telegram (1 min)

1. Open Telegram
2. Find your bot
3. Send: `/start`
4. Send: `/create warrior`
5. Send: `/help`

## You're Done! 🎉

Now play the game:
- `/stats` - Check your character
- `/mission quick` - Start a mission
- `/collect` - Claim rewards
- `/arena` - Fight other players
- `/help` - See all commands
